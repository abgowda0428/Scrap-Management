# Frontend Documentation

## Overview

The frontend is built using React and TypeScript with a clean, component-driven architecture. It focuses on usability, validation, and real-time feedback for operators and supervisors.

## Responsibilities

* UI rendering and state management
* Form validation and calculations
* Role-based UI access
* Data visualization and reports

## Key Components

* ScrapEntry
* SupervisorApproval
* JobCompletion
* CuttingOperationEntry
* ReportsEnhanced
* EndPieces & Reuse Modal
* UserManagement

## Design Principles

* No business logic in UI
* Mobile-first responsive design
* Accessibility-compliant forms
* Consistent design system

## Charts & Reports

* Built using Recharts
* Export to Excel and print support
* Date range filtering
